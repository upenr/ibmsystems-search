# Content Description
This module focus on advanced cluster features for maintaining high availability in configuration/metadata recovery, system fail-overs and system upgrades.

# Objectives
Upon completion of this course you should be able to:


Describe the concept of a quorum configuration
Identify the characteristics of the N\_Port ID Virtualization
Identify the characteristics of a hot spare/standby node
Describe the concept of node failover
Describe the benefit of performing system upgrade with spare nodes
