# Short Summary
This advanced course describes how to tune IMS databases and how to monitor and adjust items that affect performance such as VSAM, OSAM data sets, buffering, and secondary indexes.

# Content Description
This advanced level course is intended for IMS professionals getting into the tuning and performance aspects of IMS. Details include options for VSAM and OSAM, IMS access methods, and tuning database buffer pools and storage attributes.



# Objectives
You'll learn about analyzing performance data; choosing the best IMS access methods; tuning database buffer pools and storage attributes; implementing options for VSAM and OSAM; and evaluating the need for secondary indexes.
