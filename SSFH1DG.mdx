# Objectives
* Provide an overview of IBM Spectrum Fusion
* Identify the requirements for setting up IBM Spectrum Fusion
* Summarize how hardware resources on IBM Spectrum Fusion HCI are managed
* Describe encryption for storage
* Describe data protection
* Understand how to manage Call Home
* Understand how to manage Events
* Describe the process of monitoring and log collection
* Explain the IBM Spectrum Fusion user interface
* List the changes in IBM Spectrum Fusion 2.3
