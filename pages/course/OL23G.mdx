# Content Description
<p>This classroom course of two days explains how to balance the workload on the IBM i system to ensure optimum performance. Specifically, this course explains how to manage workloads, measure system performance, and tune the operating system to meet processing requirements. Hands-on exercises give you the opportunity to use the system functions that are available for controlling workload and tuning system performance. Evaluation criteria presented in this course are based on the latest information available from IBM development labs.</p>

# Objectives
<ul>
	<li>Manage jobs by:
	<ul>
		<li>Setting the appropriate system values</li>
		<li>Modifying subsystem descriptions to fit the workload</li>
		<li>Controlling batch jobs</li>
		<li>Tailoring job descriptions to fit processing needs</li>
	</ul>
	</li>
</ul>

<ul>
	<li>Create unique environments for running jobs by:
	<ul>
		<li>Creating subsystems for special applications</li>
		<li>Directing jobs to run in these subsystems</li>
		<li>Developing job descriptions for special jobs</li>
	</ul>
	</li>
</ul>

<ul>
	<li>Tune the performance of the system by:
	<ul>
		<li>Setting up pools and activity levels to handle jobs</li>
		<li>Tailoring execution parameters for job priorities</li>
		<li>Evaluating current performance using information provided by the system</li>
		<li>Changing system values and parameters as the workload changes</li>
	</ul>
	</li>
</ul>
