# Content Description
<p>Learn about Information Management System (IMS) data sharing capabilities. Examine how multiple IMS systems on different processors can access and update the same databases with complete integrity.</p>

# Objectives
<ul />
	<li />Define IMS subsystems, Internal Resource Lock Manager (IRLM), and coupling facilities to use IMS data sharing
	<li />Design recovery procedures for failures in the data sharing environment
	<li />Understand application design considerations with IMS data sharing
	<li />Monitor the performance of a data sharing environment
	<li />Explain IRLM and coupling facility roles in support of IMS data sharing


# Agenda
<ul /><li />Introduction to Data Sharing<li />Overview of Data Sharing Functions<li />Locking<li />Buffer invalidations and notifies<li />Installation and definition<li />Utilities<li />Normal operation<li />Recovery operations<li />Monitoring and diagnostic tools<li />Performance considerations<li />Batch considerations<li />Data sharing configurations and migration
