# Short Summary
This is an advanced course for IMS system programmers and database administrators, who implement and manage IMS systems using data sharing.

# Content Description
<p>Learn about Information Management System (IMS) data sharing capabilities. Examine how multiple IMS systems on different processors can access and update the same databases with complete integrity.</p>

# Objectives
<ul>
	<li>Define IMS subsystems, Internal Resource Lock Manager (IRLM), and coupling facilities to use IMS data sharing</li>
	<li>Design recovery procedures for failures in the data sharing environment</li>
	<li>Understand application design considerations with IMS data sharing</li>
	<li>Monitor the performance of a data sharing environment</li>
	<li>Explain IRLM and coupling facility roles in support of IMS data sharing</li>
</ul>

# Agenda
<ul><li>Introduction to Data Sharing</li><li>Overview of Data Sharing Functions</li><li>Locking</li><li>Buffer invalidations and notifies</li><li>Installation and definition</li><li>Utilities</li><li>Normal operation</li><li>Recovery operations</li><li>Monitoring and diagnostic tools</li><li>Performance considerations</li><li>Batch considerations</li><li>Data sharing configurations and migration</li></ul>
