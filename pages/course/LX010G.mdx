# Content Description
<p>This course is designed to teach the basics of the Linux environment. Class activities include accessing a system, navigating the directory hierarchy, using the vi editor, using common commands, tools, and scripting concepts.</p>

# Objectives
<p>On Completion of this course, the student should be able to:</p>

<ul />
	<li />Log in to an Linux system and set a user password
	<li />Use Linux online documentation
	<li />Use the vi editor
	<li />Manage Linux files and directories
	<li />Describe the purpose of the shell
	<li />Execute common Linux commands and manage Linux processes
	<li />Customize the working environment
	<li />Use common Linux utilities
	<li />Write simple shell scripts
