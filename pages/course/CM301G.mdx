# Content Description
<p>This course of 5 sessions, 8 hours each day, teaches how to tune Information Management System (IMS) databases. It also teaches IMS database options that affect performance such as data set considerations and buffering for Virtual Storage Access Method (VSAM) and Overflow Sequential Access Method (OSAM) databases.
</p><p>Practice these skills in intensive machine exercises.</p>

# Objectives
<ul><li> Analyze performance data about the IMS database environment
</li><li>Choose IMS access methods and options that provide the best database performance
</li><li>Improve performance by selecting database buffer pools and buffer pool options and with appropriate data set access method and storage attributes
</li><li>Implement the optimum performance options for VSAM data sets at define and execution time 
</li><li>Evaluate the need for secondary indexes and select options to improve their performance </li></ul>

# Agenda
<ul><li>Welcome</li><li>Introduction to IMS Database Tuning</li><li>Introduction to the Lab project</li><li>Review of Commonly Used IMS Access Methods</li><li>Measuring IMS DB Performance</li><li>Tuning VSAM Buffers</li><li>Tuning VSAM Data Sets</li><li>Additional Performance Issues</li><li>Tuning Secondary Indexes</li><li>Tuning HDAM</li><li>Tuning OSAM Data Sets and Buffers</li><li>Other Considerations</li><li>Database Tuning Summary</li></ul>

# Audience
<p>This is an intermediate to advanced course for IMS system programmers, database administrators, and technical support  individuals, who support and maintain IMS databases.</p>
