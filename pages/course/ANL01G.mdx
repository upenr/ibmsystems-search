# Content Description
<p>This course helps students with AIX basic or advanced background quickly start working in the Linux environment. AIX system concepts and working experiences are applied for transitioning to Linux environments. Hands-on lab exercises let students practice Linux commands and apply what they know from working with AIX.</p>

<p>&nbsp;</p>

# Objectives
<p>After completing this course, you should be able to:<br />
1.&nbsp;&nbsp; &nbsp;Summarize the overviews of Linux versus AIX in Power Systems<br />
2.&nbsp;&nbsp; &nbsp;Recognize the Linux operating system installation process<br />
3.&nbsp;&nbsp; &nbsp;Identify RHEL system management tools based on AIX backgrounds<br />
4.&nbsp;&nbsp; &nbsp;Summarize system configurations and devices on RHEL<br />
5.&nbsp;&nbsp; &nbsp;Identify the tasks in configuring logical partitions<br />
6.&nbsp;&nbsp; &nbsp;Carry out software installation and maintenance in RHEL<br />
7.&nbsp;&nbsp; &nbsp;Recognize RHEL TCP/IP implementation as compared to AIX<br />
8.&nbsp;&nbsp; &nbsp;Carry out system startup and shutdown in RHEL<br />
9.&nbsp;&nbsp; &nbsp;Identify Logical Volume Manager in Linux as compared to AIX<br />
10.&nbsp;&nbsp; &nbsp;Carry out basic file system administration<br />
11.&nbsp;&nbsp; &nbsp;Identify the tasks in backup and restore RHEL<br />
12.&nbsp;&nbsp; &nbsp;Summarize error monitoring and system dump processes</p>

<p>&nbsp;</p>
