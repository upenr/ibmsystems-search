# Content Description
<p>This course is designed to give the z/OS Operator personnel as well as entry systems programmers an overview about JES2. The students should be able to use JES2 commands at the z/OS Console and use SDSF.</p>

# Objectives
<ul />
	<li />Understand the z/OS environment and JES2 role
	<li />Start JES2 with focus on options and shutdown
	<li />Control JES2 with commands and SDSF
	<li />Control JES2 job flow, selection, and processing
	<li />Control output processing and printers
	<li />Understand z/OS influence on JES2 processing
