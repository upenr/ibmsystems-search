# Content Description
<p>Learn how to write application programs that use Data Language One (DL/I) to process Information Management System (IMS) databases.</p>

# Objectives
<ul>
	<li>Describe the basic differences between files and IMS databases as they relate to application program coding and logic</li>
	<li>Describe the terms and processing characteristics of IMS hierarchic data structures</li>
	<li>Construct DL/I calls for either COBOL or PL/I language programs</li>
	<li>Write, test and debug a batch application program that uses the DL/I call functions</li>
	<li>Use the IMS Test Program to prototype application program calls, and to perform ad-hoc read and update access to an IMS database</li>
	<li>Describe the applications and uses of other DL/I features</li>
	<li>Program for recovery and restart using DL/I calls to establish synchronization points</li>
	<li>Describe the functions provided by High Availability Large Database (HALDB), and any program differences when accessing a Full Function database or a HALDB</li>
</ul>

# Agenda
<ul><li>DL/I Environment</li><li>DL/I Call Processing</li><li>DL/I Rtrieval Calls</li><li>DL/I Update Calls</li><li>DL/I Programming Techniques: Part 1</li><li>DL/I Advanced Segment Search Arguments SSAs: Part 1</li><li>DL/I Test Program</li><li>DL/I Programming Techniques: Part 2</li><li>DL/I Advanced Segment Search Arguments SSAs: Part 2</li><li>System Service Calls</li><li>Programming Standards</li><li>High Availability Large Database (HALDB) Access</li></ul>

# Audience
