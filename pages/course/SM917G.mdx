# Content Description
<p>This course introduces and explains the System Automation for z/OS (SA z/OS) components, architecture, and concepts. Focus is on the System Operations component with Processor Operations as an optional topic. It is a prerequisite for attending the operations or admin classes.</p>

<p>This course does not include labs.</p>

<p>For information about other related courses, see the IBM Security Systemseducation training paths website:<br />
ibm.com/training/</p>
