# Content Description
<p>This course provides you with the knowledge to prepare your Linux on System z for production. You customize z/VM for Linux, install Linux, define network connections, and define a VSWITCH. You create different types of journal file systems and a Logical Volume Manager. A security topic addresses alternate authentication methods and explains how to harden Linux security. You employ various security measures. You create Linux clones, as well as VM shared segments for the Linux kernel. Topics also include Linux clusters and methods for backing up data. Finally, you install and use Webmin to perform administrative activities and use other performance measurement tools.</p>

# Objectives
<ul>
	<li>Customize the z/VM environment to prepare for Linux guests</li>
	<li>Install Linux in a guest virtual machine</li>
	<li>Define Linux networking configurations</li>
	<li>Define journal file systems</li>
	<li>Use LVM</li>
	<li>Setup clusters and VSwitches</li>
	<li>Define shared segments for Linux to use</li>
	<li>Clone Linux instances</li>
	<li>Provide alternate ways to authenticate Linux users</li>
	<li>Harden security on Linux on System z</li>
	<li>Share parts of the Linux file system to reduce DASD storage requirements</li>
	<li>Install and use administration tools</li>
</ul>
