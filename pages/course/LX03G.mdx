This course is for experienced Linux users to learn how to administer a Linux system.

# Content Description
<p>The purpose of this course is to provide experienced Linux users additional skills that may help them become a capable system admiistrator.</p>

# Objectives
<ul>
	<li>Install Linux from a network install server, setup network install servers, and perform automated installs</li>
	<li>Manage system and services startup and shutdown</li>
	<li>Select and use system administration tools when appropriate</li>
	<li>Configure and manage printers</li>
	<li>Use packaging tools to create, install, and de-install packages</li>
	<li>Configure and manage the X Window System and VNC</li>
	<li>Manage logging</li>
	<li>Manage kernel services</li>
	<li>Manage devices</li>
	<li>Manage hard disks, partitions, Redundant Array of Independent Disks (RAID), and Logical Volume Management (LVM)</li>
	<li>Create and manage file systems and file system quota</li>
	<li>Perform memory management</li>
	<li>Use scheduling tools</li>
	<li>Create and restore backups</li>
	<li>Perform user administration</li>
	<li>Apply user-level security</li>
	<li>Troubleshoot Linux problems</li>
</ul>
