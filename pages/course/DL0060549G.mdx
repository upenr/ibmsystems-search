Introduction into Shared Memory Communications (SMC).

# Duration in Hours ⌚

# Content Description
This course provides an introduction into Shared Memory Communications (SMC). It covers the high-level concepts of SMC and related technologies.

# Objectives
The main objectives of this course:Provide a foundation for understanding the key concepts of SMC. It introduces new terms, the high-level SMC architecture, and the basic SMC specifications (RFC).
