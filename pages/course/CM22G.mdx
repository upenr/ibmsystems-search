# Content Description
<p>Learn how to design, implement, reorganize, and recover Information Management System (IMS) databases. Practice these skills in intensive machine labs.</p>

# Objectives
<ul>
	<li>Code the Database Directories (DBD) and Program Specification Blocks (PSB) for physical databases</li>
	<li>Code the DBDs and PSBs to implement secondary indexing</li>
	<li>Use the appropriate IMS utilities to reorganize and recover physical databases, including those with secondary indexes</li>
	<li>Use the DL/I test program</li>
	<li>Prepare a job stream to load a database using different IMS access methods</li>
</ul>

# Agenda
