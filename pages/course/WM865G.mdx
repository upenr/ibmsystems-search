# Content Description
<p>In this course, you learn how to design, code, and debug command-level CICS application programs for e-business or traditional environments. The course applies to the entire family of CICS products.</p>

<p>For information about other related courses, visit the IBM Training website:<br />
http://www.ibm.com/training</p>

# Objectives
<ul>
	<li>Design and develop application programs for CICS environments</li>
	<li>Design applications with separate presentation and business logics</li>
	<li>Describe the purpose of the CICS Clients</li>
	<li>Code CICS commands</li>
	<li>Pass and receive control between program modules</li>
	<li>Describe and compare the CICS COMMAREA and the CICS channel and container interface</li>
	<li>Prepare programs and run tests, including problem determination and resolution, by using messages and online interactive debugging tools</li>
	<li>Access and modify Virtual Storage Access Method (VSAM) files, DB2 databases, temporary storage, and transient data queues</li>
	<li>Respond to normal and exception conditions from CICS commands</li>
	<li>Send and receive basic mapping support (BMS) maps</li>
</ul>
