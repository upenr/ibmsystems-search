# Content Description
<p>Learn how to code, create and maintain DB2 UDB for i physical and logical files using native Data Description Specifications (DDS) and Control Language (CL) commands. Build a DB2 UDB for i relational database foundation, if you intend to write application programs for the IBM i system, however the course does not address SQL implementation of DB2 UDB for i.</p>

<p>Learn many techniques that, if implemented, enhance the performance of applications using the database.</p>

# Objectives
<ul>
	<li>Code the DDS and create the physical and logical files necessary to implement a DB2 UDB for i relational database</li>
	<li>Describe the advantages of a field reference file, code the necessary DDS and create a field reference file</li>
	<li>Explain how the choices made while coding the DDS and creating the file affect the performance of the system and the programs using the files</li>
	<li>Describe the considerations for using the DB2 UDB for i referential integrity and triggers</li>
</ul>
