# Short Summary
Fundamental system skills in z/OS is a digital course offering which teaches the essential skills necessary to work in a z/OS environment.  This self paced course has a live lab environment.

# Content Description
<p>This course is designed to teach you the fundamental practical skills to navigate and work in a z/OS environment.&nbsp;This includes the use of ISPF/PDF dialogs, TSO/E commands, JCL, UNIX System Services shell, and BookManager.</p>

<p>The content is self-paced. The lab environment is a live lab environment, offering direct access to a IBM Z server.&nbsp;</p>

# Objectives
<ul>
	<li>Review the main concepts of z/OS</li>
	<li>Log on to TSO and start ISPF/PDF</li>
	<li>Navigate through ISPF/PDF dialogs and use the basic ISPF/PDF functions and the ISPF Editor</li>
	<li>Use ISPF/PDF to allocate data sets and edit data sets (including hierarchical file system (HFS) files) using the ISPF Editor primary and line commands</li>
	<li>Use ISPF to create and manipulate (copy, rename, delete, list, sort, and merge) data sets</li>
	<li>Identify security considerations for Multiple Virtual Storage (MVS) data sets and HFS files</li>
	<li>Describe and use TSO/E commands</li>
	<li>Perform simple modifications to existing ISPF/PDF panels</li>
	<li>Invoke a REXX exec and TSO CLIST</li>
	<li>Tailor existing JCL and submit batch jobs</li>
	<li>Review job status and job output using SDSF</li>
	<li>Invoke UNIX processes</li>
	<li>Manipulate HFS directories and file systems using the UNIX System</li>
	<li>Services ISHELL</li>
</ul>
