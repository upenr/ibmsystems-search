# Content Description
<p>This course describes the concepts and implementation details of PowerHA SystemMirror for Linux on IBM Power Systems servers. The lecture content and hands-on labs cover the basics of the product&rsquo;s architecture, prerequisites, the planning necessary for implementation, the configuration and customization steps to produce a highly available cluster, and the administration of the cluster. The course also includes a lecture-only unit describing the Smart Assists that are available, treating those products as likely/potential use cases.</p>

# Objectives
<ul>
	<li>Use the documentation for PowerHA SystemMirror for Linux for planning an implementation of the product</li>
	<li>Install prerequisites and the PowerHA SystemMirror for Linux software</li>
	<li>Create a highly available cluster using PowerHA SystemMirror for Linux using the clmgr CLI</li>
	<li>Define all resources and resource groups necessary to make a simple application highly available</li>
	<li>Verify the cluster configuration and perform cluster test operations</li>
	<li>Consider the use of PowerHA SystemMirror for Linux Smart Assists for SAP HANA and SAP NetWeaver</li>
</ul>
