IBM Spectrum Virtualize represents the software engine of IBM SAN Volume Controller  (SVC). This is a series of modules to learn and utilize this software.

# Content Description
<p>This module focus on advanced cluster features for maintaining high availability in configuration/metadata recovery, system fail-overs and system upgrades.</p>

<p>If you are enrolling in a Self Paced Virtual Classroom or Web Based Training course, before you enroll, please review the Self-Paced Virtual Classes and Web-Based Training Classes on our Terms and Conditions page, as well as the system requirements, to ensure that your system meets the minimum requirements for this course. http://www.ibm.com/training/terms</p>

# Objectives
<p>Upon completion of this course you should be able to:</p>

<p>Describe the concept of a quorum configuration<br />
Identify the characteristics of the N_Port ID Virtualization<br />
Identify the characteristics of a hot spare/standby node<br />
Describe the concept of node failover<br />
Describe the benefit of performing system upgrade with spare nodes</p>
