# Content Description
<p>This course introduces basic administrative tasks associated with Linux Servers hosted on Power Systems servers. This includes installation, command line operations, system administration, device management, IBM tools, and system recovery.</p>

# Objectives
<p>After completing this course, you should be able to:</p>

<ul>
	<li>Recognize different execution environments that a Linux server runs on within a POWER8 processor - based server</li>
	<li>Install Linux Server on a POWER8 processor - based server</li>
	<li>Manage Linux Server within a POWER8 processor - based server environment using standard tools</li>
	<li>Install and use IBM Service and Productivity tools</li>
	<li>Customize the configuration of the boot loader</li>
	<li>Use recovery mode to rescue a failed Linux instance</li>
</ul>
