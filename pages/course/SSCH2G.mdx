# Duration in Hours ⌚
8.0

# Content Description
IBM Storage Defender Data Protect provides comprehensive data backup and recovery. It provides support for a wide range of cloud-native, edge and on-premise workloads at scale. Within this it addresses governance and compliance requirements for data protection and incident response, thereby significantly reducing risk.

This one-day course will focus on the creation and initial maintenance of a Data Protect cluster. The hands-on exercises are performed on a Linux environment.

# Objectives
• Deploy an IBM Data Protect cluster

• Use the local administrative interfaces

• Understand how to add further nodes to the cluster

• Know how to perform a basic upgrade of a Data Protect cluster

# Agenda
• Unit 1: Introduction to IBM Data Protect

• Unit 4: Hardware setup guide

- LAB1: Deploy three nodes and create a Data Protect cluster

• Unit 7: Organizations and multi-tenancy

• Unit 8: Integration with Storage Protect

• Unit 9: Cluster upgrade and update

- LAB2: Add a cluster node and upgrading the cluster
