# Content Description
<p>This course is an overview of the z/OS operating systems and services for non-MVS IS technical personnel.</p>

# Objectives
<ul>
	<li>Identify the installation requirements for an enterprise server and how z/OS supports these requirements</li>
	<li>Discuss the services for application enablement in a z/OS environment</li>
	<li>Identify how z/OS provides support for On Demand Business</li>
	<li>Describe how security is ensured in a z/OS environment</li>
	<li>Identify products and tools that support a central point of control</li>
	<li>Characterize products and facilities that support z/OS systems</li>
	<li>Describe the essential system services of z/OS</li>
	<li>List the z/Architecture processor configurations for z/OS</li>
	<li>Describe z/OS connectivity, communication facilities, and interfaces</li>
	<li>Describe how zEnterprise creates a single system of systems</li>
	<li>The concept, strategy, and benefits of the z/OS environment</li>
	<li>Application enablement in z/OS</li>
	<li>Overview of On Demand Business support in z/OS</li>
	<li>Connectivity to the z/OS environment</li>
	<li>Security support provided by z/OS</li>
	<li>Systems Management support (Tivoli, SMP/E, SMF, and RMF)</li>
	<li>Scalability, availability, backup, and recovery features in z/OS</li>
	<li>System Services support (storage management, job management, work management, data sets, and data set management)</li>
	<li>System z processor configurations, System z Enterprise Server hardware</li>
	<li>zEnterprise, a new dimension in computing</li>
</ul>

# Agenda
<ul><li>Unit 1: Requirements of an enterprise server</li><li>Unit 2: Application enablement</li><li>Unit 3: Enabling On Demand Business on z/OS</li><li>Unit 4: z/OS connectivity</li><li>Unit 5: Security in a z/OS environment</li><li>Unit 6: Systems management</li><li>Unit 7: System and data availability</li><li>Unit 8: System services</li><li>Unit 9: System z9 and z10 and System z hardware</li><li>Unit 10: zEnterprise: A new dimension in computing</li></ul>

# Audience
<p>This basic course is for personnel with a technical background in information technology who are new to the z/OS platform and do not have prior MVS or OS/390 experience.</p>
