# Content Description
This intermediate course describes how to build COBOL and PL/I applications that access IMS databases, specifically, how to construct DLI calls, test, and debug applications.



# Objectives
You'll learn about IMS hierarchic databases; DL/I calls for COBOL or PL/1; debugging batch applications; using the IMS DL/I test program; programming for recovery and restart; programming differences between HALDB and full-function databases.
