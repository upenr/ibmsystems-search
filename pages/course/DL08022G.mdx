# Content Description
Red Hat OpenShift Container Storage is software-defined storage integrated with and optimized for containers. OpenShift Container Storage is built on Red Hat Ceph Storage, Rook, and NooBaa to provide container native storage services that support block, file, and object services. OpenShift Container Storage will deploy anywhere OpenShift does: on-premises or in the public cloud.

OpenShift Container Storage has taken the most common tasks, from deployment through to day-to-day operations, while emphasizing greater scalability and deeper integration with OpenShift to:
*	Deploy storage services
*	Expand storage
*	Provide out-of-the-box dashboards in OpenShift Administrator Console to indicate health and status of the storage
