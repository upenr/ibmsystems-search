# Content Description
<p>The ZV45 course is designed to provide students that are responsible for the administration of the KVM for IBM z System product with the knowledge to install, manage, and configure the product.&nbsp; You will leave the course with the knowledge to implement KVM for IBM z System, apply a service pack, add new disk storage to the KVM server, deploy Linux guests, and migrate deployed guests to other KVM server.</p>

# Objectives
<p>&nbsp;&nbsp;&bull; Install KVM for IBM z System</p>

<p>&nbsp; &bull; Apply a fixpack for KVM for IBM z System</p>

<p>&nbsp; &bull; Initial storage devices that are used by KVM for IBM z System</p>

<p>&nbsp; &bull; Manage network devices and create an OpenSystem vSwitch</p>

<p>&nbsp; &bull; Deploy SLES12 guest with KVM for IBM z System</p>

<p>&nbsp; &bull; Convert deployed guests from a qcow2 image to a raw image</p>

<p>&nbsp; &bull; Resize the deployed guest</p>

<p>&nbsp; &bull; Migrate KVM guests from one KVM server to another.</p>
