# Content Description
<p>Learn how to transfer your generic UNIX system administration skills to the administration of Fedora and Red Hat Enterprise Linux and/or SUSE Linux.</p>

<p>This course includes lab exercises to reinforce what was taught in lecture. This course helps the student to prepare for the Linux Professional Institute (LPI) Certification.</p>

# Objectives
<ul>
	<li>Discuss the history of Linux</li>
	<li>Perform a network and automatic installation</li>
	<li>Use Linux online documentation</li>
	<li>Understand and manipulate Linux startup and shutdown flow</li>
	<li>Identify system administration tools</li>
	<li>Identify and use utilities to install and update software packages</li>
	<li>Identify and use common Linux block devices</li>
	<li>Describe, create, and use Linux filesystems</li>
	<li>Create and manage Linux users and groups</li>
	<li>Learn common troubleshooting methods</li>
	<li>Describe and use memory management utilities</li>
	<li>Describe and configure virtualization using Xen</li>
</ul>
