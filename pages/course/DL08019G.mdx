# Short Summary
The DS8900F is the eighth generation of the DS8000 family. It's built on the same market-proven, reliable code base. This course provides an overview of the key capabilities of the DS8900F.

# Duration in Hours
3

# Content Description
The DS8900F is the eighth generation of the DS8000 family. It's built on the same market-proven, reliable code base. It maintains the same key capabilities as its predecessors, including cyber resilience and security, replication and multi-site topologies, ultra low latency, and high performance.


# Objectives
*	Identify the core themes of the DS8900F family
*	Summarize the features and functions of the DS8900F
*	Distinguish the benefits of the three DS8900F models
*	Describe the RAS functions of the DS8900F
*	Distinguish those elements that contribute to virtualization for the DS8900F
