# Content Description
<p>This course is designed to prepare students to install and configure a highly available cluster using PowerHA SystemMirror.</p>

# Objectives
<ul /><li />Explain what high availability is<li />Outline the capabilities of PowerHA SystemMirror<li />Design and plan a highly available PowerHA cluster<li />Install and configure PowerHA SystemMirror in the following modes of operation:<li />Single resource group on a primary node with standby node (active/passive)<li />Two resource groups in a mutual takeover configuration (active/active)<li />Configure resource group startup, fallover, and fallback policies<li />Perform basic system administration tasks for PowerHA<li />Perform basic customization for PowerHA<li />Perform basic problem determination and recovery<p>This course is the first in the PowerHA 7 curriculum. # Audience
<p>The audience for this advanced course is students who are experienced AIX system administrators with TCP/IP networking <strong>and</strong> AIX Logical Volume Manager (LVM) experience who are responsible for the planning and installation of a PowerHA SystemMirror 7.1 <strong>and</strong> later cluster on an IBM Power Systems server running AIX V6.1 or later. The lab exercises are conducted on an AIX 7.1 TL2 level system.</p>
