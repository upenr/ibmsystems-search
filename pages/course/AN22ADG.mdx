# Content Description
<p>This online course is supplemental to <u>(AN22G) AIX Network Installation Management Concepts and Configuration</u>.</p>

<p>The new NIM features in AIX 7.2 are highlighted over other features. For example, AIX 7.2 introduces Live Update as a major feature that puts new requirements on using NIM. But, one of the key NIM features is to use HTTP in place of NFS. This course takes students through configuration, limitation, and use cases for working with NIM over HTTP.</p>

<p>This course puts students in the new AIX 7.2 environment, without going through the installation or upgrading processes that are described with hands-on activities in <u>AN22G</u>.</p>
