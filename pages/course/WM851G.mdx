# Content Description
<p>An updated version of this course is available. For more information, click <em>CICS V5.3 Fundamentals </em>(WM806G).</p>

<p>In this entry-level course, you learn about the basic concepts and facilities of IBM CICS Transaction Server V5. After completing this course, you have the prerequisite knowledge for all of the follow-on CICS application and systems programming courses that IBM Training offers.</p>

http://www.ibm.com/training</p>

# Objectives
<ul>
	<li>Describe the role of CICS in a business transaction environment</li>
	<li>Outline the flow of a typical CICS transaction</li>
	<li>Describe the functions that the primary CICS-supplied transactions perform</li>
	<li>Use the CICS master terminal transaction (CEMT) to query and control system resources</li>
	<li>Describe various ways of defining resources in CICS</li>
	<li>Use transactions that CICS supplies to define and manage resources</li>
	<li>Explain how CICS stores and manages resource definitions</li>
	<li>Describe the features of the CICS Explorer</li>
	<li>List the facilities that CICSPlex System Manager provides</li>
	<li>Describe how CICS implements security</li>
	<li>List the recoverable resources of CICS</li>
	<li>Describe CICS transaction backout and forward recovery</li>
	<li>List the types of CICS startup and shutdown</li>
	<li>Describe the facilities that CICS provides for maintaining the integrity of data that is associated with CICS programs</li>
	<li>Describe CICS support for databases and Virtual Storage Access Method (VSAM) files</li>
	<li>Describe the functions of presentation, business, and data access logic</li>
	<li>Define the role and use of the COMMAREAs and CHANNELs in CICS applications</li>
	<li>Show how CICS Basic Mapping Support (BMS) supports the 3270 display</li>
	<li>Describe the syntax of a CICS command</li>
	<li>Explain how to handle exceptions</li>
	<li>Use the transactions that CICS supplies for testing and debugging</li>
	<li>Describe the facilities that are available in CICS for intercommunication</li>
</ul>
