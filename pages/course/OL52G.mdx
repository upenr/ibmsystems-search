# Content Description
<p>This course is designed to familiarize the student with functions of the HMC using lecture and lab exercises.</p>

# Objectives
<ul>
	<li>Install an HMC and go through the setup wizard</li>
	<li>Connect the HMC to a managed system</li>
	<li>Create and manage partitions manually or with a .sysplan</li>
	<li>Manage the HMC including updating code for the HMC and firmware in the managed system</li>
</ul>
