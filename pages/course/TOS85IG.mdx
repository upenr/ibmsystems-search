# Content Description
<p>Learn to install and use FlashCopy Manager 4.1 for Windows to protect Windows file systems, Microsoft Exchange, and Microsoft SQL Server. This course covers information about protecting data by using local VSS snapshots and sending snapshots to Tivoli Storage Manager 7.1. There are no hands-on exercises, but video demonstrations of the tasks are included.</p>

<p>If you are enrolling in a Self Paced Virtual Classroom or Web Based Training course, before you enroll, please review the <strong>Self-Paced Virtual Classes and Web-Based Training Classes</strong> on our Terms and Conditions page, as well as the system requirements, to ensure that your system meets the minimum requirements for this course.</p>

<p>http://www.ibm.com/training/terms</p>

# Objectives
<ul>
	<li>Describe Windows Volume Snapshot Service (VSS) snapshot architecture for protecting applications and file systems</li>
	<li>Install FlashCopy Manager for Windows</li>
	<li>Protect, back up, and restore Windows file systems using VSS snapshots</li>
	<li>Protect, back up, and restore Microsoft SQL Server using VSS snapshots</li>
	<li>Protect, back up, and restore Microsoft Exchange using VSS snapshots</li>
</ul>

# Agenda
<ul><li>Overview and installation</li><li>Windows file systems and SQL Server</li><li>Microsoft Exchange</li><li>PowerShell, remote management, and scheduling</li><li>FlashCopy Manager integration</li></ul>

# Audience
<p>This intermediate-level course is for Storage Administrators for FlashCopy Manager for Windows who are familiar with storage concepts and work with other storage experts (database administrators, system administrators, mail administrators, etc.) to protect applications by using FlashCopy Manager and storage snapshot technology.</p>
