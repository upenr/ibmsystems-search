# Content Description
In this course, you will use both the Operations Center and command line to define, modify, and add retention holds to retention rules and retention sets.

# Objectives
• Explain how Retention Sets and Retention Holds protect data • Differentiate between Retention Rules and Retention Sets • Create a Retention Rule with the Operations Center • Modify and delete Retention Rules with the Operations Center • Generate an on-demand Retention Set • Add a Retention Hold to a Retention Set • Copy a Retention set to tape
