# Content Description
<p>This course is designed to describe the features and functions of Fast Path both from a capability and an implementation perspective.</p>

# Objectives
<ul>
	<li>Identify the components of IMS Fast Path</li>
	<li>Describe the features and functions of each component</li>
	<li>Explain the benefits and restrictions of each component</li>
	<li>Describe the processing of Fast Path resources within the IMS TM and DB Control environment</li>
	<li>Design and implement an Expedited Message Handler (EMH) environment</li>
	<li>Design and implement Fast Path databases (DEDBs and MSDBs)</li>
	<li>Implement High Speed Sequential Processing (HSSP)</li>
	<li>Implement the DEDB Virtual Storage Option</li>
	<li>Plan and implement the conversion of MSDBs to VSO-DEDBs</li>
	<li>Develop an effective buffer pool strategy</li>
	<li>Implement database and system recovery in a Fast Path environment</li>
	<li>Implement data sharing for Fast Path DEDBs</li>
	<li>Understand the benefits of Shared EMH</li>
	<li>Write application calls to Fast Path databases</li>
	<li>Design operator procedures for managing Fast Path resources</li>
</ul>

# Agenda
<ul><li>Overview of IMS Fast Path</li><li>Expedited Message Handler (EMH)</li><li>Main Storage Database (MSDB)</li><li>Data Entry Database (DEDB)</li><li>Fast Path System Functions</li><li>Parallel Sysplex Exploitation</li><li>Fast Path Administration</li><li>Fast Path Operations</li><li>Application Programming and Design</li></ul>

# Audience
<p>This is an advanced course for IMS application designers, application programmers, system administrators, and database administrators, who design or implement IMS Fast Path applications in an IMS Transaction Management (TM) or CICS Database Control (DBCTL) environment.</p>
