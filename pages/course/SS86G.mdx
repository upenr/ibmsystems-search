# Content Description
<p>This course prepares you to implement and manage DFSMShsm (DFSMS hierarchical storage manager). You will learn how to automate the storage management tasks, which optimize DASD space utilization and provide data availability. You will be taught basic DFSMShsm terminology and concepts, and you will examine the functions and interfaces used to tailor DFSMShsm processing to meet your installation&#39;s storage management requirements.</p>

<p>Hands-on lab exercises enable you to perform a step-by-step implementation.</p>

# Objectives
<ul>
	<li>Understand the relationship between DFSMShsm and DFSMS and get an overview of the functions of DFSMShsm and its operating environment.</li>
	<li>Understand the DFSMShsm activities for Primary and Secondary Space Management and setup the required configuration for automatic space management.</li>
	<li>Authorize special users for DFSMShsm commands and perform space management functions using authorized DFSMShsm commands.</li>
	<li>Understand the DFSMShsm activities for Availability Management and setup the required configuration for automatic availability management.</li>
	<li>Implement the Automatic Backup and Recovery Support (ABARS), and perform data set and volume management using authorized DFSMShsm commands.</li>
	<li>Understand DFSMShsm processing in large systems.</li>
	<li>Understand recovery using ABARS, recovery from data set or volume loss, recovery of control data sets and journals.</li>
	<li>Use the AUDIT command to detect and fix discrepancies and the LIST, QUERY and FIXCDS commands to correct error situations.</li>
	<li>Manage and monitor DFSMShsm using log data sets and statistical information as well as manage DFSMShsm tapes.</li>
</ul>
