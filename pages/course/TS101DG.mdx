# Content Description
<p>The IBM Spectrum Protect (formerly Tivoli Storage Manager or TSM) suite of products&nbsp;<br/>includes the Spectrum Protect Server and many add-on products and components for&nbsp;<br/>providing enterprise class data protection and disaster recovery. In this Fundamentals&nbsp;<br/>of Spectrum Protect course we provide a simple, easy-to-understand introduction to many&nbsp;<br/>of the components and technologies associated &nbsp;with Spectrum Protect along with a series&nbsp;<br/>of videos to help better understand the content.</p><p>&nbsp;</p>

# Objectives
<p>1. Understand the core data storage and protection concepts that IBM Spectrum Protect is based on</p><p>2. Navigate the main IBM Spectrum Protect command line and GUI interfaces</p><p>3. Perform basic data backups and differentiate between backup types, such as incremental backups&nbsp;</p><p>and complete backups</p><p>&nbsp;</p>
