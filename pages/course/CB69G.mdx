# Content Description
<p>This course is designed to provide the skills for basic implementation and customization of TCP/IP on z/OS.</p>

# Objectives
<ul>
	<li>Describe IBM&#39;s z/OS TCP/IP product offering</li>
	<li>Describe the installation requirements and system customization for the z/OS TCP/IP implementation</li>
	<li>Define network interface attachments that produce connectivity for z/OS TCP/IP communications</li>
	<li>Construct TCP/IP configuration statements for z/OS to allow operation and connectivity to an existing TCP/IP network</li>
	<li>Configure TCP/IP for z/OS for basic communications functions</li>
	<li>Construct TCP/IP configuration statements for z/OS for selected TCP/IP application protocols services</li>
</ul>
