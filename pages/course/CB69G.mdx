# Content Description
<p>This course is designed to provide the skills for basic implementation and customization of TCP/IP on z/OS.</p>

# Objectives
<ul />
	<li />Describe IBM&#39;s z/OS TCP/IP product offering
	<li />Describe the installation requirements and system customization for the z/OS TCP/IP implementation
	<li />Define network interface attachments that produce connectivity for z/OS TCP/IP communications
	<li />Construct TCP/IP configuration statements for z/OS to allow operation and connectivity to an existing TCP/IP network
	<li />Configure TCP/IP for z/OS for basic communications functions
	<li />Construct TCP/IP configuration statements for z/OS for selected TCP/IP application protocols services
