# Content Description
<p>This course will teach you to implement and customize RACF on z/OS.</p>

<p>We will start creating and maintaining the RACF databases. The functions, a resource manager can use, to work with RACF as external security manager are discussed. A short overview about the RACF profiles and the administration of them is given.</p>

<p>RACF customization, like the class descriptor table and started procedure table, are discussed. This also includes performance and tuning of RACF.</p>

<p>Program control and the interaction between the storage management components and RACF is presented.</p>

<p>An overview about multilevel security is given.</p>

# Objectives
<ul><li>Create and maintain the RACF database</li><li>Customize RACF</li><li>Maintain RACF implementation</li><li>Protect system resources</li></ul>
