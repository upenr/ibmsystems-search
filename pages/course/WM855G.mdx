# Objectives
<ul>
	<li>Install CICSPlex System Manager</li>
	<li>Describe and use the web user interface (WUI) provided by CICSPlex System Manager</li>
	<li>Configure and define CICSplexes and associated CMAS environments by using the WUI</li>
	<li>Define and use the CICS Explorer configured with the CMCI connected to a CICSplex</li>
	<li>Define CICS systems to CICSPlex System Manager by using the WUI and CMCI</li>
	<li>Manage the operations of a CICSplex of CICS systems by using the WUI and CMCI</li>
	<li>Create definition records to be used as directives by CICSPlex System Manager applications, such as Workload Manager, real-time analysis, and Monitor Services</li>
	<li>Use CICSPlex System Manager (CPSM) commands to view and dynamically manage the data that CICSPlex System Manager applications use</li>
	<li>Use the BAS to create and manage CICS resources through application scopes</li>
	<li>Use the WUI editor facilities to define WUI views, menus, users, and user groups, and to edit favorites</li>
	<li>Describe the CPSM API and BATCHREP</li>
	<li>Describe and implement the CICSPlex SM security facilities</li>
	<li>Describe how to use the CICS Explorer to deploy a CICS platform and application to the cloud</li>
</ul>
