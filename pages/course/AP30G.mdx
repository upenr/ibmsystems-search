# Content Description
<p>IBM Cloud PowerVC Manager, a cloud offering in the PowerVC family, provides a self-service cloud portal for IBM Power Systems and is built on OpenStack. It also includes all the functions of the IBM PowerVC Standard Edition. It provides an easy way to provision and manage AIX or Linux virtual machines on PowerVM based systems in a private or hybrid cloud setting. This course aims to deliver an overall understanding as well as basic practical skills for PowerVC products.</p>

# Objectives
<ul>
	<li>Summarize the PowerVC virtualization management functions</li>
	<li>Recognize PowerVC hardware and software requirements</li>
	<li>Implement PowerVC software installation and upgrade</li>
	<li>Recognize the use of PowerVC user interfaces</li>
	<li>Recall PowerVC administration tasks in working with the environment</li>
	<li>Recognize configurations of resources, projects, and users</li>
	<li>Recall PowerVC procedures in capturing virtual machines</li>
	<li>Recognize PowerVC capabilities of virtual image deployment</li>
	<li>Summarize scenarios of multi-tenancy in PowerVC with projects</li>
	<li>Demonstrate an understanding of cloud computing from a technology standpoint</li>
	<li>Summarize the architecture of IBM Cloud PowerVC Manager</li>
	<li>Integrate IBM Cloud PowerVC Manager with an existing virtual infrastructure</li>
	<li>Demonstrate an ability to access and interact with the graphical and command line interfaces of the product</li>
	<li>Summarize the concepts of creating new projects, users, networking configuration and other cloud related properties</li>
	<li>Summarize the key administrative tasks that need to be performed after the installation of IBM Cloud PowerVC Manager</li>
</ul>
