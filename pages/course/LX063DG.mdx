# Content Description
<p>This course is designed to provide skills enablement for system administrators and product support specialists in the area of operating system-based virtualization provided by Linux containers on IBM Power.</p><p>&nbsp;</p><p>Hands-on exercises reinforce the lecture material, allowing students to install and configure Linux containers.</p>

# Objectives
<p>After completing this course, you should be able to:</p><ul /><li />Describe Linux containers architecture<li />Install and configure Docker onto a Power Systems environment<li />Perform Docker administration operations<li />Install and configure CRI-O<li />Perform basic container orchestration steps
