# Content Description
<p>This course presents the IBM Transformation Advisor on OpenShift on Power Systems, which helps administrators and developers to modernize and migrate their applications from on-premises environments to the cloud. &nbsp;<br /><br />Every application is different in terms of structure, configuration, and goals. It typically requires an in-depth analysis of the applications. IBM Transformation Advisor assists you in this process by discovering, assess, and plan for migration of these applications to the new environment.</p>

# Objectives
<p>After completing this course, you should be able to answer the following questions:</p><ul /><li />What is Transformation Advisor, why use it, and when using it?<li />What are the features of Transformation Advisor, and how to use it?<li />Where to deploy Transformation Advisor?
