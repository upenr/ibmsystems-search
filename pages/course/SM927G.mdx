# Content Description
<p>This course introduces and explains the System Automation for z/OS&reg; (SA z/OS) commands that are used for system operations. In this course, the System Automation for z/OS automation manager and automation agent run in a z/OS 2.2 environment. The automation platform, Tivoli&reg; NetView for z/OS is at version 6 release 2. The course uses several automation scenarios in single and multisystem configurations to demonstrate the concepts that are taught in the lessons. This training class is delivered in an environment with multiple opportunities for hands-on lab exercises.</p>

# Objectives
<p>When you complete this course, you can use IBM System Automation for z/OS 4.1 or IBM Automation Control for z/OS to perform the following tasks:</p>

<ul />
	<li />Start and stop automated resources
	<li />Monitor and control automated resources
	<li />IPL and shut down systems and an SAplex
