This course introduces the IBM  Cloud Paks, enterprise-grade container software designed to offer a faster, more reliable way to build, move, and manage on the cloud.

# Content Description
<p>This 8-hour, self-paced course provides students with a review of containers and Kubernetes in an OpenShift environment as preparation for a deeper look at IBM Cloud Pak offerings. &nbsp;The student learns about the components of each of the Cloud Paks available on Power Systems, the common services that are part of all IBM Cloud Paks, and then what is necessary to implement Cloud Paks. &nbsp;The course closes with pointers to additional resources for the student to use as the next step in their journey to IBM Cloud Paks on Power Systems.</p>

# Objectives
<p>After completing this module, you should be able to:</p><ul /><li />Recall containers and Kubernetes concepts and basic capabilities.<li />Recall the manageable resources of the Red Hat Openshift platform and its role in a hybrid cloud environment.<li />Describe the goals, benefits, and design of IBM Cloud Paks on Power Systems.<li />Verify the foundation layers and capabilities of the IBM Cloud Paks on a Red Hat Hybrid Cloud Architecture.<li />List and discuss the IBM Cloud Platform Common Services incorporated in several IBM Cloud Paks.<li />List and discuss the current available IBM Cloud Pak solutions.<li />Identify the components and capabilities included in each IBM Cloud Pak.<li />Identify the necessary steps to install an IBM Cloud Pak.<li />List the IBM Cloud Pak related publications.

# Agenda
<p>This course is bundled into 5 units:</p><p>&nbsp;</p><p>1. Kubernetes and Red Hat OpenShift Container Platform Review<br />2. IBM Cloud Paks Foundation<br />3. IBM Cloud Pak Solutions<br />4. Installing an IBM Cloud Pak Software Example<br />5. IBM Cloud Pak Related Publications</p>
