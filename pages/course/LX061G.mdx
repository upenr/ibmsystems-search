# Content Description
<p>This course is designed to provide skills enablement for system administrators and product support specialists in the area of operating system based virtualization provided by Docker. Topics include introduction to Docker, planning the installation, Docker administration, and using images and containers. Hands-on exercises reinforce the lecture material, and allow students to install and configure Docker.</p>

# Objectives
<p>On completion of this course, students should be able to:</p>

<ul>
	<li>Describe the architecture of the Docker platform</li>
	<li>Install and configure Docker onto a Power Systems environment</li>
	<li>Perform Docker administration operations</li>
	<li>Perform user operations with images and containers</li>
	<li>Understand the concept of Docker orchestration</li>
</ul>

# Agenda
<ul><li>Unit 1: Introduction to Docker</li><li>Exercise 1: Accessing the lab environment</li><li>Unit 2: Implementing Docker on IBM Power Systems</li><li>Exercise 2: Docker installation</li><li>Unit 3: Docker images and containers</li><li>Exercise 3: Docker images and containers</li><li>Unit 4: Container orchestration</li></ul>

# Audience
<p>This course is for anyone that needs to build experience with operating system based virtualization using Docker. The audience for this training includes the following:</p>

<ul>
	<li>Cloud administrators</li>
