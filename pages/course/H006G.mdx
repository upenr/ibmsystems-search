# Content Description
<p>The course includes information on the Spectrum Scale installer toolkit, installer GUI, cluster upgrade scenarios, cluster health monitoring, cluster performance data collection and monitoring, management GUI, file system encryption, cluster security, and call home features. Many of the features are described in lecture materials and implemented in lab exercises.</p><p>&nbsp;</p>
