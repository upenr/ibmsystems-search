# Short Summary
This course discusses installing, updating, and removing RPM packages from the IBM AIX Toolbox for Open Source Software.

# Duration in Hours ⌚
4.0

# Content Description
This course enables students to install and manage open source software on the AIX operating system. This includes installing, updating, and removing RPM packages, from the IBM AIX Toolbox for Open Source Software.

# Objectives
* Describe the benefits of open source software on AIX
* Describe the IBM AIX Toolbox for Open Source Software
* Understand the different methods for managing open source software on AIX (rpm and DNF)
* Learn how to install DNF and manage repositories on AIX
* Perform installation and management of open source packages on AIX

# Agenda
* Unit 1: Open source software on AIX
* Exercise 1: The AIX Toolbox and working with RPM packages in the lab
* Unit 2: Installing DNF and managing repositories on AIX
* Exercise 2: Installing DNF and configuring local repositories
* Unit 3: Installation and management of open source packages on AIX
* Exercise 3: Package management with DNF

# Audience
This course is intended for anyone who requires knowledge and skills to install and manage open source software on AIX 7.3.
