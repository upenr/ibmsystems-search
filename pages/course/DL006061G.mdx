# Short Summary
This course explores topics focused on new and interesting features IBM Z Security and Resiliency.

# Duration in Hours ⌚
16.0

# Content Description
By completing this in-person course, passing the topic specific quizzes and passing the final exam, students will be eligible to earn the IBM Z Security and Resiliency Essentials badge. Earners are expected to have a basic understanding of the topics as well as where to find more information to continue learning and growing their proficiency.

# Objectives
This badge earner has demonstrated an understanding on how IBM's mainframe security and resiliency solutions can help clients protect their environments, as well as features and integrations that will enhance the value of these IBM solutions. The earner is equipped with the understanding of key segments related to security and resiliency and how the IBM Z Security solutions will aid clients in improving their strategies for those segments.
