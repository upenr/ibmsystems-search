# Content Description
<p>This course will teach you how to use shell scripts and utilities for practical system administration of AIX (or other UNIX) operating systems.</p>

# Objectives
<p>After completing this course, you should be able to:<br />
&bull; Distinguish Korn and bash shell specific features<br />
&bull; Use utilities such as sed and awk to manipulate data<br />
&bull; Understand system shell scripts such as /etc/shutdown<br />
&bull; Write useful shell scripts to aid system administration</p>

# Agenda
<ul><li>Basic shell concepts</li><li>Flow control in a shell Script</li><li>Functions and typeset</li><li>Shell features such as arithmetic and string handling</li><li>Using regular expressions</li><li>Using sed, awk and other AIX utilities</li></ul>
