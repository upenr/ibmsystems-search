# Content Description
<p>This is a four hour, intermediate level, self paced learning module that describes IBM PowerVC, benefits, how to transition to using it, and how it improves Power Systems virtualization management.</p>

# Objectives
<p>Module 1</p>

<ul>
	<li>Describe the history of PowerVM management</li>
	<li>List common LPAR management tasks</li>
	<li>Describe how the common LPAR management tasks are performed using the Hardware Management Console (HMC)</li>
</ul>

<p>Module 2</p>

<ul>
	<li>Describe the historical challenges of creating a traditional PowerVM workload</li>
</ul>

<p>Module 3</p>

<ul>
	<li>Compare the process of creating a PowerVM workload in PowerVC with that of the traditional HMC-based process</li>
</ul>

<p>Module 4</p>

<ul>
	<li>Compare traditional PowerVM management tasks with the PowerVC equivalent tasks
	<ul>
		<li>Review examples PowerVM management tasks and their PowerVC equivalents</li>
		<li>Creating a PowerVM LPAR using PowerVC</li>
		<li>Start an LPAR</li>
		<li>Stop an LPAR</li>
		<li>Perform DLPAR on an LPAR</li>
		<li>Perform LPM on an LPAR</li>
		<li>Delete an LPAR</li>
	</ul>
	</li>
</ul>

<p>Module 5</p>

<ul>
	<li>Outline the benefits of introducing PowerVC management</li>
	<li>Review the requirements to implement PowerVC</li>
	<li>Outline the future of PowerVM management with PowerVC</li>
	<li>Explain the next steps on the PowerVC Learning Journey</li>
</ul>

<p>&nbsp;</p>
