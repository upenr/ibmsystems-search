# Content Description
<p>This course is an overview of the z/OS operating systems and services for non-MVS IS technical personnel.</p>

# Objectives
<ul />
	<li />Identify the installation requirements for an enterprise server and how z/OS supports these requirements
	<li />Discuss the services for application enablement in a z/OS environment
	<li />Identify how z/OS provides support for On Demand Business
	<li />Describe how security is ensured in a z/OS environment
	<li />Identify products and tools that support a central point of control
	<li />Characterize products and facilities that support z/OS systems
	<li />Describe the essential system services of z/OS
	<li />List the z/Architecture processor configurations for z/OS
	<li />Describe z/OS connectivity, communication facilities, and interfaces
	<li />Describe how zEnterprise creates a single system of systems
	<li />The concept, strategy, and benefits of the z/OS environment
	<li />Application enablement in z/OS
	<li />Overview of On Demand Business support in z/OS
	<li />Connectivity to the z/OS environment
	<li />Security support provided by z/OS
	<li />Systems Management support (Tivoli, SMP/E, SMF, and RMF)
	<li />Scalability, availability, backup, and recovery features in z/OS
	<li />System Services support (storage management, job management, work management, data sets, and data set management)
	<li />System z processor configurations, System z Enterprise Server hardware
	<li />zEnterprise, a new dimension in computing
