# Short Summary
Learn to design, deploy, and manage IBM Power Virtual Server workloads on IBM Cloud, covering core concepts, networking, storage, and migration to optimize PowerVS for business needs.

# Duration in Hours ⌚
8.0

# Content Description
This course is designed for IT architects, specialists, developers, sales professionals, and anyone interested in implementing workloads on IBM Power using the IBM Cloud platform. It explores the core concepts, management techniques, networking protocols, storage mechanisms, practical use cases, and migration strategies of IBM Power Virtual Server (PowerVS).

By the end of the course, participants will have a strong foundation and hands-on knowledge to confidently navigate the complexities of IBM PowerVS and optimize its capabilities to meet their specific business requirements.

# Objectives
* Outline the scope and key elements of the IBM Cloud platform
* Describe the capabilities of the IBM Power Virtual Server offering
* Identify and describe the PowerVS framework
* List the key terminologies and management entities
* Interpret networking configuration
* Provision workspace, virtual servers, storage, and network interfaces
* Explain methods and tools on how to manage PowerVS resources
* Plan migration to PowerVS
