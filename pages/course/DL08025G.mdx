# Content Description
This course covers the foundations of Artificial Intelligence (AI) and Big Data and how they relate to IBM Spectrum Storage. Students are introduced the basic concepts of Machine Learning (ML), Deep Learning (DL), and popular ML frameworks, such as TensorFlow and Pytorch.

Following this overview, students will learn about the IBM Spectrum Storage offerings and how they provide support for AI and Big Data projects.
                            
After completing the course, students will earn the IBM Spectrum Storage for AI and Big Data Fundamentals badge: https://www.credly.com/org/ibm/badge/ibm-spectrum-storage-for-ai-and-big-data-fundamentals 

# Objectives
* Recall the basic concepts of Artificial Intelligence, Machine Learning, and Deep Learning
* Distinguish between the various Machine Learning/Deep Learning frameworks
* Summarize the ways in which IBM's products for Data and AI improve the AI process
