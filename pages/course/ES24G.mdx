# Content Description
<p>This course teaches you navigation, operation, and recovery techniques for the Hardware Management Console (HMC) in the System z environment. Through lecture and hands-on exercises, you learn setup and operating procedures for the HMC, gain in-depth problem determination skills, practice HMC operations, and utilize recovery capabilities provided by the System z servers.</p>

# Objectives
<p>Identify the zEnterprise 196 and 114, System z10, System z9, and zSeries servers</p>

<p>Describe the major differences of the function and features between the System z servers</p>

<p>Describe and explain the concept of logical partitioning</p>

<p>Identify how various resources are assigned to LPARs during initialization</p>

<p>Describe the functions provided by the HMC and SE</p>

<ul>
	<li>User interface styles, local and remote connectivity</li>
</ul>

<p>Identify HMC home page to log on HMC application</p>

<p>Use the classic UI to:</p>

<ul>
	<li>Log on and log off</li>
	<li>Open objects</li>
	<li>Navigate the HMC workplace using various selection techniques</li>
	<li>Exit a task on a window</li>
	<li>View the online books and use the Help functions</li>
	<li>Identify the different types of groups
	<ul>
		<li>CPC group, control hardware</li>
		<li>Images group, control operating system</li>
	</ul>
	</li>
	<li>Build new CPC and images groups</li>
	<li>Identify the contents of the Details window and determine object status</li>
	<li>Identify the various task lists</li>
</ul>

<p>Use the tree UI to:</p>

<ul>
	<li>Log on and log off</li>
	<li>Open objects</li>
	<li>Navigate the HMC workplace using various selection techniques</li>
	<li>Exit a task on a panel</li>
	<li>View the online books and use the Help functions</li>
	<li>Identify the different types of groups
	<ul>
		<li>CPC group, control hardware</li>
		<li>Images group, control operating system</li>
	</ul>
	</li>
	<li>Build new CPC and images groups</li>
	<li>Identify the contents of the Details window and determine object status</li>
	<li>Identify the various task lists</li>
</ul>

<p>Describe how exceptions are reported and presented on the HMC using the tree or classic UI</p>

<p>Use the HMC to determine the type and cause of the exception</p>

<p>Describe the three types of hardware messages that can surface on the HMC</p>

<p>Use the HMC to determine the cause of the hardware message</p>

<p>Retrieve the hardware messages from the log</p>

<p>Use the HMC Operating System Messages task to display and send messages to the operating system</p>

<p>Describe the functions that require the use of the access administrator ID</p>

<p>Use the HMC to perform access administrator related tasks</p>

<p>Describe the purpose, functions, and use of the different types of activation profiles</p>

<ul>
	<li>Reset profile</li>
	<li>Image profile</li>
	<li>Group profile</li>
	<li>Load profile</li>
</ul>

<p>Use the HMC to build new profiles, modify, view, or delete existing ones</p>

<p>Use the HMC to determine or change profile assignments Provide operational recommendations for HMC user settings</p>

<p>Use the HMC to perform</p>

<ul>
	<li>A manual load operation</li>
	<li>A system reset</li>
	<li>Problem determination for a load failure</li>
</ul>

<p>Invoke the system activity display and describe its purpose</p>

<p>Establish an HMC/SE session using the Single Object Operations task</p>

<p>Identify the difference between the HMC and SE workplace</p>

<p>Navigate using classic or tree UI to Identify CP, channel, and CHPID status for the physical CPC and all LPs</p>

<p>Identify the entry point for I/O and Channel Problem Determination panels</p>

<p>Log off from a CPC session</p>
