# Content Description
<p>Learn about a key availability and capacity enhancement to Information Management System (IMS) - the Shared Queues facility, which offers the ability to share full function message queues within a parallel sysplex environment. Also, learn about sharing Fast Path Expedited Message Handler (EMH) queues within a parallel sysplex environment.</p>

# Objectives
<ul><li>Describe the hardware and software components of IMS Shared Queues
</li><li>Understand how the initial IMS configuration impacts planning for the implementation of Shared Queues
</li><li>Identify the key migration and operational considerations of IMS Shared Queues implementation</li></ul>

<p>This is an intermediate course for experienced IMS programmers, IMS Database (DB) and Transaction Manager (TM) system administrators, IMS application designers, and IMS operations individuals who plan, implement, and support IMS Shared Queues.</p>
