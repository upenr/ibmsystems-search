# Content Description
<p>This course is designed to prepare you to install and configure a highly available cluster using PowerHA SystemMirror.</p>

# Objectives
<ul>
	<li>Explain what high availability is</li>
	<li>Outline the capabilities of PowerHA SystemMirror</li>
	<li>Design and plan a highly available PowerHA cluster</li>
	<li>Install and configure PowerHA SystemMirror in the following modes of operation:
	<ul>
		<li>Single resource group on a primary node with standby node (active/passive)</li>
		<li>Two resource groups in a mutual takeover configuration (active/active)</li>
	</ul>
	</li>
</ul>

<ul>
	<li>Configure resource group startup, fallover, and fallback policies</li>
	<li>Perform basic system administration tasks for PowerHA</li>
	<li>Perform basic customization for PowerHA</li>
	<li>Perform basic problem determination and recovery</li>
</ul>

# Audience
<p>The audience for this course are those who are experienced AIX system administrators with TCP/IP networking and AIX Logical Volume Manager (LVM) experience who are responsible for the planning and installation of a PowerHA SystemMirror 7.1 and later cluster on an IBM Power Systems server running AIX 5L V5.3 or later. The lab exercises are conducted on an AIX 6.1 TL6 -level system.</p>
