# Content Description
<p>This course is designed to provide you with the skills required to install and customize z/OS UNIX (full name z/OS UNIX System Services), and to manage and monitor the z/OS UNIX environment.</p>

# Objectives
<ul>
	<li>Execute the tasks required to prepare a z/OS installation for implementing z/OS UNIX</li>
	<li>Execute the tasks to install the z/OS UNIX software features</li>
	<li>Use the information provided in this class to perform the basic customization necessary to fully implement the z/OS UNIX kernel, the file system, the shell and utilities, and z/OS UNIX applications</li>
	<li>Put in place the RACF security required for z/OS UNIX resources and applications</li>
	<li>Make appropriate definitions for the activation of TCP/IP sockets by z/OS UNIX</li>
	<li>Identify and use the processes and data required for monitoring and tuning the z/OS UNIX environment</li>
</ul>
