# Content Description
<p>This course teaches IBM Spectrum LSF 10.1 version.</p>

<p>The course is designed to give system administrators the knowledge required to implement and maintain LSF in their working environment. They will gain a solid understanding of workload resource management, cluster configuration and administration. The workshops provide valuable experience with the installation of LSF, cluster configuration and administration. The system administrator will also learn helpful hints and tips and develop fundamental troubleshooting skills.</p>

# Objectives
<ul>
	<li>Define concepts and terms use in IBM Spectrum LSF</li>
	<li>Install IBM Spectrum LSF</li>
	<li>Submit jobs in IBM Spectrum LSF</li>
	<li>Manage resources</li>
	<li>Use cluster query commands</li>
	<li>Describe IBM Spectrum LSF architecture and configuration</li>
	<li>Describe LSF scheduling policies</li>
	<li>Describe resources allocation limits</li>
	<li>Detail LSF cluster management</li>
	<li>Troubleshoot and debug IBM Spectrum LSF</li>
	<li>Describe and work with EGO</li>
	<li>Explain application integration and deployment</li>
</ul>

# Agenda
<p>LSF concepts and terminology<br />LSF installation<br />Job submission and control<br />Resource management<br />Cluster query commands<br />LSF architecture<br />LSF configuration<br />LSF batch system configuration<br />LSF scheduling policies and fairshare policies<br />Resource limits<br />Cluster management<br />Basic troubleshooting<br />LSF debugging<br />IBM Spectrum LSF with EGO<br />LSF application integration and deployment</p>
