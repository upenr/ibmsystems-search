# Content Description
<p>This course provides an overview of the PowerVM edition&rsquo;s features on POWER6 and POWER7 processor-based systems. It explains the new features and benefits of virtualization including processor virtualization, Integrated Virtual Ethernet, Virtual I/O Server, and virtual devices, such as virtual Ethernet, virtual SCSI, and virtual Fibre Channel adapters. Basic and advanced configurations of the Virtual I/O Server and its clients are discussed including various availability options.</p>

<p>Students are also given additional details about PowerVM features that were introduced in <em>Power Systems for AIX I: LPAR Configuration and Planning (AN110)</em>.</p>

# Objectives
<ul>
	<li>Discuss the advantages or value of PowerVM edition&#39;s features</li>
	<li>Define micro-partitioning and shared processor LPARs</li>
	<li>Discuss the benefits of simultaneous multithreading</li>
	<li>Discuss and configure the Integrated Virtual Ethernet (IVE)</li>
	<li>Install and configure the Virtual I/O Server</li>
	<li>Configure virtual network devices, such as virtual Ethernet and shared Ethernet adapters</li>
	<li>Configure virtual SCSI and virtual Fibre Channel storage adapters</li>
	<li>Configure virtual SCSI target devices on a virtual SCSI adapter</li>
	<li>Define file-backed storage pools and file-backed virtual optical devices</li>
	<li>Identify single points of failure in virtualized environments</li>
	<li>Configure multiple VIO servers for high availability</li>
	<li>Configure advanced virtual networking options</li>
	<li>Configure the shared Ethernet adapter failover feature</li>
	<li>Configure advanced virtual SCSI options</li>
	<li>Configure MPIO in a VIO server&#39;s client partition</li>
	<li>Manage the service events, configure call home, add, exchange FRUs, and discuss FSP failover</li>
</ul>

# Prerequisites
<p>You must have advanced system administration experience with AIX 6 <strong>or</strong> AIX 7. This prerequisite can be met by attending one of the following courses:</p>

<ul>
	<li><em>Power Systems for AIX II: Implementation and Administration (AN120) </em></li>
	<li><em>Power Systems for AIX III: Advanced Administration and Problem Determination (AN150) </em></li>
	<li><em>AIX Jumpstart for UNIX Professionals (AN140) </em></li>
</ul>

<p>Alternatively, you must have equivalent AIX <strong>and</strong> LPAR skills.</p>

<p>General TCP/IP knowledge is strongly recommended. This prerequisite can be met by attending <em>TCP/IP for AIX Administrators (AN210). </em></p>

<p>You are also expected to have logical partition administration skills on POWER6 <strong>or </strong>POWER7 processor-based systems, which can be obtained by attending Power Systems for<em> AIX I: LPAR Configuration and Planning (AN110).</em></p>
