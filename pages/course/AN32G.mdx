# Content Description
<p>This course provides the implementation and management skills to create Virtual I/O Server (VIOS) storage groups and shared storage pools on IBM Power Systems. Students learn to create a VIOS storage group, manage storage in the shared storage pool, provision storage to client partitions using either thick or thin provisioning, mirror the shared storage pool, create and use snapshots, and manage the VIOS storage group configuration by learning how to do backups, restores, and software upgrades.</p>

# Objectives
<ul>
	<li>Configure a Virtual I/O Server storage group and a shared storage pool</li>
	<li>Manage physical volumes in the shared storage pool</li>
	<li>Create and manage logical units in the shared storage pool</li>
	<li>Allocate storage to client partitions using thick or thin provisioning</li>
	<li>Configure the shared storage pool alert thresholds</li>
	<li>Mirror the shared storage pool</li>
	<li>Create, use, and manage storage snapshots</li>
	<li>Back up and restore a VIOS storage group configuration</li>
</ul>
