# Content Description
<p>Database Control (DBCTL) is an Information Management System (IMS) facility that provides CICS access to an IMS database subsystem. DBCTL provides access for CICS transactions to IMS managed databases. Learn the activities required to install, tailor, maintain and support an IMS DBCTL system. Additionally, learn about aspects of the IMS architecture that apply to database management and attaching to CICS systems and transactions.</p>

# Objectives
<ul><li>Understand DBCTL in an IMS environment
</li><li>Design, install, and customize a DBCTL system
</li><li>Perform basic system administration for an IMS system with DBCTL</li></ul>
