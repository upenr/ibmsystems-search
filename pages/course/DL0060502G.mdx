# Content Description
ABO Essentials 102 provides guidance for using ABO, from preparing a PoC either on-premise or on cloud trial, to deploying the optimized modules to the production environment. After taking this course, for further tools that can be leveraged with ABO, explore [ABO Essentials 103: ABO native and other supplementary tools](https://learn.ibm.com/course/view.php?id=13449).



# Objectives
After taking this course, you should be able to:

* Prepare an ABO PoC
* Optimize Enterprise COBOL Modules with ABO
* Analyze the performance improvement
* Plan deployment strategy
* Troubleshoot common issues
