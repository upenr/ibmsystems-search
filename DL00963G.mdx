# Content Description
The course covers IMS Database Access Methods, the IMS Hierarchic Direct Database, Secondary Indexes, as well as the IMS Data Entry Database (DEDB), sequential segments, layout examples and database corruption. It also addresses verifying full-function database integrity, verifying DEDB database integrity, and IMS database repair techniques.Â 



# Objectives
You will learn how to:

* Identify causes and types of corruption in IMS full-function, DEDBs, and HALDB databases
* Verify database integrity by using IMS database utilities
* Repair databases by running pointer checkers
