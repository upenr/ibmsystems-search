# Content Description
This course will provide a technical introduction to deployment and operation of the on-premises IBM Cloud Object Storage system. Topics covered include an overview of the product, installation, configuration, operations, monitoring, and troubleshooting. This course does not include hands-on labs.



# Objectives
* Understand the benefits of IBM Cloud Object Storage
* Plan the architecture and configuration of IBM Cloud Object Storage
* Know how to install IBM Cloud Object Storage and perform initial configuration of an on-premises object storage vault
* Perform IBM Cloud Object Storage daily administrative tasks
* Monitor and troubleshoot an IBM Cloud Object Storage on-premises system
